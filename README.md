# Oh No, Emergency! (Date Exit Messages)

Small mobile-first web app to simulate an incoming emergency message to help you politely exit an awkward date.

Disclaimer: This app generates simulated messages only. Do NOT use it to contact real emergency services.

Setup

1. Install dependencies

```bash
npm install
```

2. Start dev server

```bash
npm run dev
```

What you'll find

- `src/data/templates.ts` — seed templates (15+)
- `src/components` — React components
- LocalStorage persistence for sessions and history

Tests: `npm run test` (uses Vitest)
